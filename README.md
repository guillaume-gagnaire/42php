42php
=====

**42php** is a simple framework to create websites faster than ever. It comes 
with admin panel, easily upgradable, some UI libs like Slick for slideshows, 
Foundation 5 for templating, responsive, etc ...

Librairies and technologies
---------------------------

* PHP >= 5.4
* MySQL
* Foundation 5 (http://foundation.zurb.com/)
* jQuery (https://jquery.com/)
* AngularJS (https://angularjs.org/)
* Foundation Icons (http://zurb.com/playground/foundation-icon-fonts-3)
* Pickadate (http://amsul.ca/pickadate.js/)
* Slick (http://kenwheeler.github.io/slick/)
* Facebook PHP SDK (https://github.com/facebook/facebook-php-sdk-v4)
* Google API PHP SDK (https://github.com/google/google-api-php-client)

See the [wiki](https://gitlab.com/alduro-estallas/42php/wikis/home) for usage.
